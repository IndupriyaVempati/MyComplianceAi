Quality Assurance Checklist

  • All HTML files render correctly in Chrome, Firefox, and Safari
  • All colours match approved hex codes exactly (no approximations)
  • Poppins font is properly loaded and displays correctly across devices
  • Fallback fonts are defined in the font stack
  • All text/background combinations pass WCAG 2.1 AA minimum (4.5:1 normal, 3:1 large)
  • Primary Accent (#FFC20E) is only used as a background or icon fill — never as text on white
  • Neutral (#CFECFB) is only used as a background — never as text colour
  • Typography hierarchy is consistent across all pages
  • Responsive design tested on mobile (375px), tablet (768px), and desktop (1280px)
  • All interactive elements have visible focus states for keyboard navigation
  • Traffic light colours are used consistently for status indicators
  • CSS variables are used throughout — no hardcoded colour values in component CSS
